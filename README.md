# 3_bars
1.Выводит название самого большого и маленького баров из входящего json файла с сайта http://data.mos.ru/opendata/7710881420-bary
2.Выводит название ближайшего бара, ввод координат, от которых отталкиваться в рассчетах, производится пользователем с клавиатуры.

Сперва пользователь должен ввести путь до файла, после чего получит имена самого большого и самого маленького бара.
После потребуется ввести координаты для поиска ближайшего бара.
